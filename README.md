# Projeto-Final---An-lise-de-Dados-Educacionais---Ensino-M-dio-Brasileiro
Análise de dados educacionais do ensino médio brasileiro utilizando dados públicos para explorar padrões de desempenho, calcular médias e gerar visualizações. Projeto desenvolvido por Alice Pereira do Nascimento e Nathalia de Almeida Costa.
