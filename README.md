# 📊 Projeto Final — Análise de Dados Educacionais do Ensino Médio Brasileiro

## 📌 Descrição
Este projeto tem como objetivo realizar uma análise de dados educacionais do ensino médio brasileiro a partir de dados públicos. A análise busca identificar padrões de desempenho, calcular médias de notas ao longo dos anos e gerar visualizações que auxiliem na compreensão do desempenho educacional entre municípios.
